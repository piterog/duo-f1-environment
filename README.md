# Duo-F1

> **EN**

This project originated from my passion for Formula 1 and the lack of a real-time assistant to provide data, numbers, and statistics about drivers and cars. The main goal is to be used as a companion, whether you're at home watching the race on TV or at the racetrack itself. Additionally, it functions as a portfolio to demonstrate my technical skills with main technologies I usually wotk with, as well as a personal development challenge.

> **PT**

O projeto nasceu da paixão de assistir a Formula 1 e da falta de ter um assistente para conferir em tempo real, dados, numero e estatisticas de pilotos e carros.
O objetivo principal é ser usado como acessório, para você que está em casa vendo a corrida pela TV ou você que está no Autódromo.
Também serve como vitrine das minhas skills tecnicas envolvendo as principais tecnologias que eu costumo trabalhar e serve como um desafio pessoal.

## How to start / installation

You will need to clone all below projects in the same folder.

- [duo-f1-backend](https://github.com/piterog/duo-f1-backend)
- [duo-f1-frontend](https://github.com/piterog/duo-f1-frontend)
- [duo-f1-stream](https://github.com/piterog/duo-f1-stream)

Go to the root folder, give execution permission to the `setup.sh` script using:

`chmod +x ./setup.sh`

Then, execute the install script using:

`./setup.sh`
